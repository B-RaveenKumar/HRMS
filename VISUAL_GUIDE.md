## 🎨 Design System

### Typography Hierarchy
```
H1: Hospital Dashboard (2.0rem, Bold)
   Main page title

H2: Doctor Dashboard (1.8rem, Bold)
   Section titles

H5: Patient Records (1.25rem, Bold)
   Card headers

H6: Total Patients (0.875rem, Bold)
   Stat card labels

Body: Regular text (1rem, Normal)
   Content text

Small: Helper text (0.875rem, Muted)
   Description text
```

### Spacing Scale
```
Extra Large:  40px (mb-4)
Large:        30px (mb-3)
Medium:       20px (mb-2)
Small:        10px (mb-1)
Compact:      5px
```

### Shadow System
```
Shadow-sm:   0 2px 4px rgba(0,0,0,0.08)    Light
Shadow-md:   0 4px 12px rgba(0,0,0,0.12)   Medium
Shadow-lg:   0 8px 24px rgba(0,0,0,0.15)   Dark
```
